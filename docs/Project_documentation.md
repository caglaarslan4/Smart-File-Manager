# Smart File Manager

## Project Description

Smart File Manager is a terminal-based application written in C.
It allows users to perform basic file operations.

## Functions

### showMenu()
Displays the main menu and the available file operations.

### createFile()
Creates a new file.

### readFile()
Reads and displays the contents of a file.

### writerFile()
Writes user input to a file.

### renameFile()
Rename an existing file.

### deleteFile()
Deletes a selected file.

### searchFile()
Searches for a file and checks whether it exists.
